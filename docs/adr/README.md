# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the KubeAuto Day IDP.

ADRs document significant technology and pattern choices made during the build.
Each ADR is written at the point of decision and committed alongside the component
it documents.

## Format

We use [MADR](https://adr.github.io/madr/) (Markdown Any Decision Records) format.
See `template.md` for the template.
